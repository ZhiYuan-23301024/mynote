### 移除元素

旨在基于原数组本身进行移除特定的某个或某几个元素

常用双指针
- 快指针 fast : 去前面找能被留下的元素
- 满指针 slow : 把fast找到的那些能留下的给留下
- 循环条件：快指针是否溢出

**LC.27题解**
```c
class Solution {

public:

    int removeElement(vector<int>& nums, int val) {

        int amount = 0;

        if (nums.size() == 0) {

            return 0;

        }

        int fast = 0;

        int slow = 0;

        while (fast != nums.size()) {

            if (nums[fast] != val) {

                nums[slow] = nums[fast];

                slow++;

                fast++;

                amount++;

            }

            else if (nums[fast] == val) {

                fast++;

            }

        }

        return amount;

    }

};
```

#### 滑动窗口

双指针的一种变种
旨在让快指针不断后移，慢指针视情况决定是否移动
**重点在于慢指针如何移动**
以及边界条件的判断

```
class Solution {

public:

    int minSubArrayLen(int s, vector<int>& nums) {

        int n = nums.size();

        if (n == 0) {

            return 0;

        }

        int ans = INT_MAX;

        int start = 0, end = 0;

        int sum = 0;

        while (end < n) {

            sum += nums[end];

            while (sum >= s) {

                ans = min(ans, end - start + 1);

                sum -= nums[start];

                start++;

            }

            end++;

        }

        return ans == INT_MAX ? 0 : ans;

    }

};
```

本过程中，end最多比start落后一个身位，并且身位落后刚好就是跳出循环的等价条件，在新循环开始时，end和start同位
