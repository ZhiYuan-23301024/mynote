## 链式表示

### 单链表

动态分配内存，节点不连续地存在堆空间中

**规定头指针为虚拟指针**


**结点定义**
```c++
# 结点定义
typedef struct Node {
	int data;
	struct Node *next;
} Node;
```

**初始化链表**
```c++
# 初始化链表
Node* init_linkedlist() {
	Node *head = (Node *)malloc(sizeof(Node)); //创建头结点
	if (head == NULL) exit(1); //内存分配失败
	head->next = NULL; //初始化为空链表
	return head;
}
```

**判断是否为空**
```c++
# 判断是否为空链表
bool is_empty(Node *head) {
	return head->next == NULL;
}
```

bool insert_at_index(int index, Node *head, Node *newNode) {
	Node *cur = head;
	if (index < 0) return false;
	while (int i = 0; i < index; i++) {
		if (cur->next == NULL) {
			return false;
		}
		cur = cur->next;
	}
